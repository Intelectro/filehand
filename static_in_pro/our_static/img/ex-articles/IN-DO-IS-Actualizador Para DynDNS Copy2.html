Metodo de instalacion de Raspbian

  • La tarjeta micro SD debe estar en formato FAT (minimo 4GB de memoria , clase 10 preferentemente)
  • Descargar la ultima version de noobs (incluye raspbian, version lite no lo incluye)Noobs (instalador)
  • Extraer los achivos en la raiz de la micro sd Intoducir micro SD en Raspberry y encenderlo (solo se conecta a una toma de corriente)
  • Selecciona el SO (sistema operativo), en este caso Raspbian ya incluido, o descargar uno via internet.
  • Presionar en INSTALL, al terminar la instalacion aparecera mensaje "install complete" dar enter y reiniciara el raspberry.
  • Despues del reinicio se mostrara la pantalla de escritorio del raspberry.

Instalacion de ddclient (raspbian)

(Las imagenes que se muestran son para complementar la explicacion)

  • Se debe tener una conexion a internet
  • Ejecutar en la cosola de comandos: sudo apt-get install ddclient , el cual descargara e instalara ddclient nos dara 2 opciones S/N (espanol) o Y/N (ingles), escribimos S o Y ,segun sea el caso terminara de instalar y aparesera una interfaz para ingresar nuestros datos
  • CON LAS FLECHAS DEL TECLADO ubicar www.dyndns.com y dar enter para seleccionar.

 

  • INGRESAR USUARIO: 081bfQ-dyndynamicuser

 

  • INGRESAR PASWORD: superR12 (puede requierir ingresarlo nuevamente)

 

  • Nos dara una pregunta para utilizar checkip.dyndns.com para verificar IP (seleccionamos SI o YES, segun sea el caso)

 

  • Metodo de seleccion de nombres (seleccionar De una lista y dar enter, cargara los nombres a actualizar, con la tecla espaciadora se coloca un asterisco indicando que se selecciono, dar enter para confirmar.

 

  • Preguntara si desea ejecutar ddclient al establecer una conexionPPP (seleccionamos NO y dar enter)

 

  • Preguntara si desea ejecutar ddclient como un demonio en el arranque del sistema (seleccionar SI)

 

  • Ingresar el tiempo para la comprobacion de la direccion de interfaz ( ejemplos: 5s (segundos), 3m (minutos), 7h (horas) o 1d (dia)

 

  • Editar el archivo de configuracion (Con interfaz o Manual)
  • (Reconfigurar ddclient con Interfaz) en la consola de comandos ingresar: sudo dpkg-reconfigure ddclient (nos aparesera la interfaz para reconfigurar algun que se necesite modificar)
  • (Modificar el archivo de configuracion Manualmente) en la consola de comandos ingresar: sudo leafpad /etc/ddclient.conf ( para modificar el archivo de configuracion manualmente)
